# Stock Alarm 개인정보 처리방침 초안

시행 예정일: 2026-05-18

이 문서는 App Store와 Play Store 제출을 준비하기 위한 개인정보 처리방침 초안입니다. 실제 배포 전에는 운영 주체, 연락처, 공개 URL, 서버 운영 방식, 제3자 처리 현황을 다시 확인한 뒤 게시해야 합니다.

## 1. 서비스 개요

Stock Alarm은 사용자가 입력한 보유 종목, 평단가, 수량, 알림 기준을 바탕으로 가격 조건과 배당 일정을 알려주는 앱입니다. 앱은 투자 자문, 매매 주문, 증권 계좌 연결을 제공하지 않습니다.

현재 MVP는 사용자가 직접 실행하는 로컬 서버 또는 사용자가 입력한 Stock Alarm 서버와 연결됩니다. 계정 가입은 없으며, 앱은 익명 기기 ID와 기기 비밀값으로 사용자의 기기를 구분합니다.

## 2. 수집 또는 저장하는 정보

| 구분 | 항목 | 목적 |
|---|---|---|
| 기기 식별 정보 | 익명 기기 ID, deviceSecret, 기기 표시 이름, 플랫폼 | 기기별 데이터 분리와 모바일 API 인증 |
| 푸시 정보 | Expo Push Token | 가격 조건 도달, 테스트 푸시, 배당 일정 알림 전송 |
| 사용자가 입력한 금융 정보 | 종목 코드, 표시 이름, 계좌 구분, 증권사/계좌명, 평단가, 보유 수량, 매수일, 알림 기준, 투자 메모, 매수 이유, 목표가, 매도 조건 | 포트폴리오 표시, 계좌별 보유분 분리, 알림 기준 계산, 사용자 메모 관리 |
| 배당 정보 | 주당 연 배당금, 배당 주기, 지급월, 배당락일, 지급일, 변경 이력 | 배당 캘린더와 예상 현금흐름 표시 |
| 앱 활동 정보 | 알림 이력, provider 진단 로그, 백업 메타데이터 | 알림 확인, 장애 진단, 데이터 복구 |

Stock Alarm은 주민등록번호, 생체정보 원본, 증권 계좌 비밀번호, 주문 권한 정보를 요구하지 않습니다.

## 3. 정보 이용 목적

- 사용자가 등록한 종목과 알림 기준 저장
- 가격 조건 도달 여부 계산
- 배당 일정과 예상 배당 현금흐름 표시
- 텔레그램 또는 모바일 푸시 알림 전송
- 시세/배당 provider 장애 진단
- 백업과 복구 제공

## 4. 제3자 제공 또는 처리

| 대상 | 전달될 수 있는 정보 | 목적 |
|---|---|---|
| Expo Push service | Expo Push Token, 알림 제목과 본문 | 모바일 푸시 알림 발송 |
| Telegram Bot API | 알림 메시지, 종목명, 가격, 알림 기준 | 텔레그램 알림 발송 |
| 시세/배당 provider | 종목 코드, 회사명 후보 | 시세와 배당 정보 조회 |

GitHub 저장소에는 코드와 문서만 올라가야 하며, `.env`, 봇 토큰, API 키, `data/store.json`, 백업 파일은 업로드하지 않습니다.

## 5. 보관 기간

로컬 MVP에서는 데이터가 사용자의 PC 또는 사용자가 지정한 서버의 저장소에 보관됩니다. 사용자가 종목을 삭제하거나 백업 파일을 삭제하면 해당 항목은 앱 저장소에서 제거됩니다.

향후 공용 서버로 운영할 경우, 기기 연결 해제와 서버 데이터 삭제 절차를 앱 안에서 제공하고 이 문서에 보관 기간을 명확히 표시해야 합니다.

## 6. 사용자 선택권

사용자는 다음 작업을 할 수 있습니다.

- 종목 등록, 수정, 삭제
- 종목별 알림 켜기/끄기
- 모바일 푸시 권한 허용 또는 거부
- 앱의 기기 연결 해제
- 웹 관리자 화면 또는 텔레그램 명령어를 통한 백업 생성, 복구, 삭제

현재 앱의 `기기 연결 해제`는 앱 내부에 저장된 익명 기기 정보를 지우는 기능입니다. 공용 서버 제출 전에는 서버에 저장된 기기 데이터와 푸시 토큰까지 삭제하는 기능을 별도로 제공해야 합니다.

## 7. 보안

- deviceSecret은 앱 내부 안전 저장소에 보관합니다.
- 서버에는 deviceSecret 원문 대신 해시를 저장합니다.
- `.env`, API 키, 텔레그램 봇 토큰, 로컬 데이터 파일은 GitHub에 올리지 않습니다.
- 공용 서버 운영 시 HTTPS를 사용해야 합니다.

## 8. 아동 개인정보

Stock Alarm은 아동을 대상으로 설계된 앱이 아닙니다. 실제 스토어 제출 시 대상 연령과 금융 정보 표시 방식에 맞춰 연령 등급 설문을 정확히 작성해야 합니다.

## 9. 시세와 투자 판단 고지

Stock Alarm이 표시하는 시세와 배당 정보는 provider 업데이트 시점에 따라 지연되거나 누락될 수 있습니다. 앱은 투자 자문 또는 매매 권유를 제공하지 않으며, 모든 투자 판단과 매매 실행은 사용자 본인의 책임입니다.
