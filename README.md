# Stock Alarm

Stock Alarm tracks the highest price after a user registers a stock and sends repeated sell alerts when the current price drops by a configured percentage from that high.

## Initial MVP

- Register watched stocks
- Track the highest price after registration
- Send Telegram alerts when the price drops by a configured percentage
- Keep sending alerts until the user disables or updates the condition

## Long-Term Goal

- Launch on the Apple App Store and Google Play Store
- Support account-based watchlists and push notifications
- Provide stable stock price data integration

## Suggested Build Order

1. Telegram-based MVP
2. Web dashboard for stock and alert management
3. Mobile app with push notifications
4. Store release and operations tooling
